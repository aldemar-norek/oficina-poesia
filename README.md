# Oficina de Poesia com Nuno Rau

Landing page para a Oficina de Poesia do Instituto Estação das Letras.

## Deploy em Vercel

### Opção 1: Deploy Automático (Recomendado)

1. Acesse https://vercel.com/new
2. Clique em "Continue with GitHub" (ou crie conta)
3. Autorize Vercel a acessar seu GitHub
4. Clique em "Import Git Repository"
5. Cole esta URL: https://github.com/seu-usuario/oficina-poesia
6. Clique em "Import"
7. Vercel fará o deploy automaticamente

Seu site estará em: `https://oficina-poesia.vercel.app`

### Opção 2: Deploy via CLI

```bash
npm i -g vercel
vercel
```

Siga as instruções na tela.

### Opção 3: Deploy via Drag & Drop

1. Acesse https://vercel.com/new
2. Arraste a pasta `oficina_poesia_deploy` para a área indicada
3. Clique em "Deploy"

Pronto! Seu site estará online em segundos.

## Arquivos

- `index.html` - Página principal
- `style.css` - Estilos
- `script.js` - Interatividade
- `assets/` - Imagens
- `vercel.json` - Configuração Vercel

## Customização

Para editar conteúdo, altere `index.html` e faça commit no Git. Vercel fará redeploy automaticamente.
